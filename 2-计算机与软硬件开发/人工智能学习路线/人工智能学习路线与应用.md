#### libcudart.so not found
I have fixed it by including the .dll and fixed the file path. It now works on windows 10.<br />[https://github.com/Keith-Hon/bitsandbytes-windows.git](https://github.com/Keith-Hon/bitsandbytes-windows.git)<br />Install the bitsandbytes library by
```python
pip install bitsandbytes-windows.
```
Be noted that it may not work directly with transformers library as it references the bitsandbytes package by using 'bitsandbytes' name. <= to avoid this issue, you could directly install from the git repo
```python
pip install git+https://github.com/Keith-Hon/bitsandbytes-windows.git
```
